# OS8 Roadmap

## Vision

Create an extremely useful and fun personal operating system. Agents have their own distinct personalities and know each user's unique preferences. Together, they create any app the user wants with ease — limited only by imagination.

## Near-term Focus

- **Agent Management** — Build a dedicated system for managing agents, making it easier for users to leverage advanced features and customize their agents
- **Documentation** — Create onboarding docs with screenshots and videos that help new users understand how to use the system
- **Install Testing** — Validate install packages across different machines and platforms

## Medium-term Focus

- **Sharing on os8.ai** — Create a seamless way for users to share their agents and apps with the community
- **Advanced App Building** — Enhance the build system to support more complex plans and task lists for longer, multi-step app development
- **Multi-agent Research** — Explore more complex multi-agent setups (hierarchies, delegation, specialized roles) for future consideration

## Community

OS8 is a young open-source project. We're actively engaging with personal agent and app-building communities to spread the word, encourage participation, and gather feedback.

We welcome feature requests, bug reports, and ideas — open an [issue](https://github.com/os8ai/os8/issues) or start a [discussion](https://github.com/os8ai/os8/discussions).
